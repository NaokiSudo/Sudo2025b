This folder provides R scripts and data for Sudo(2025) published in SSM Population Health.

1_Figure2.R: make Figure 2.

2_analysis.R: make Table 3, Table 4 (upper side), and Table 6.

3_analysis.R: make Table 2, Table 4 (under side), and Table 5.

4_supply1.R: make Table S1.

5supply_2.R: make Table S2

kaken2025new.csv: experimental data

The SSP2022 data is available upon request to the SSP Project (http://ssp.hus.osaka-u.ac.jp/). 

Sudo, Naoki. 2025. ``Subjective, Intersubjective, and Objective Social Statuses: How do People Imagine Social Inequality?'',
SSM - Population Health, in press, https://doi.org/10.1016/j.ssmph.2025.101878.
